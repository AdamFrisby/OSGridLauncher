OSGridLauncher by AdamFrisby

The OSGrid Launcher Application

Automatically download the latest OSGrid OpenSimulator version and run a region on OSgrid.org. Handles firewall & router configuration automatically.

Dependencies

Mono.Nat SharpZipLib .NET or Mono
